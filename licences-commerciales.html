CAPE

Cadre d'Analyse Parentale par Évaluation

Framework mathématique d'aide à la décision judiciaire en matière de garde d'enfants. Adaptation du ThibScore GKSC pour objectiver l'évaluation parentale selon l'Article 33 du Code civil du Québec (intérêt supérieur de l'enfant).

Clientèle Cible

Tribunaux québécois (Cour supérieure, Cour du Québec) • Cabinets d'avocats (droit familial) • Travailleurs sociaux (DPJ, CISSS) • Experts judiciaires (psychologues, évaluateurs)

  • Méthodologie GKSC adaptée : 4 composantes (Géométrique, Cinématique, Spectroscopique, Contextuel) appliquées à l'évaluation parentale
  • Scoring quantitatif : Réduction de la subjectivité judiciaire par grille d'analyse standardisée
  • Conformité juridique : Aligné sur l'Article 33 du Code civil du Québec
  • Documentation scientifique : Rapport détaillé avec méthodologie reproductible
  • Formation incluse : 8h de formation personnalisée pour juges/avocats/travailleurs sociaux
  • Support continu : Assistance méthodologique 12 mois (email + téléphone)

NTVC

Network Threat Vector Coefficient

Framework de détection d'anomalies multi-dimensionnelle pour cybersécurité. Adaptation du ThibScore GKSC pour analyse d'événements réseau et réduction massive des faux positifs (-73% projection).

Clientèle Cible

Grandes entreprises (500+ employés) • Institutions financières (banques, assurances) • Gouvernements (ministères, agences) • SOC (Security Operations Center)

  • Analyse GKSC temps réel : 4 composantes + 12 sous-variables pour scoring d'anomalies
  • Validation empirique : R² = 0.85 (framework astrophysique), transférable à la cybersécurité
  • Intégration SIEM : API REST, Python SDK, webhooks (Splunk, ELK, QRadar)
  • Dashboard interactif : Application web complète avec présets de test
  • Déploiement flexible : Cloud (AWS, Azure, GCP) ou On-Premise
