JPL/NASA

Jet Propulsion Laboratory - Small-Body Database

Base de données de référence mondiale pour les objets du système solaire : comètes, astéroïdes, TNOs, NEAs. Idéal pour objets interstellaires confirmés.

Types d'observations acceptées

  • Objets interstellaires (1I, 2I, 3I...)
  • Trajectoires hyperboliques
  • Paramètres orbitaux précis
  • Données photométriques
Small-Body Database Browser HORIZONS System CNEOS (Météores interstellaires)

MPC

Minor Planet Center (IAU)

Centre officiel de l'Union Astronomique Internationale pour la désignation et le suivi des objets mineurs : astéroïdes, comètes, TNOs, objets interstellaires.

Soumissions scientifiques

  • Observations astrométriques
  • Nouvelles découvertes
  • Corrections d'orbites
  • Rapports détaillés

MUFON

Mutual UFO Network

Plus grande organisation mondiale d'investigation ufologique. Collecte systématique de témoignages, analyse scientifique, base de données des observations UAP/UFO.

Observations UAP acceptées

  • Témoignages visuels
  • Photos/vidéos
  • Traces physiques
  • Données radar

Centre américain de collecte d'observations UFO/UAP. Base de données publique consultable en ligne, hotline téléphonique 24/7, archivage systématique.

Rapports UFO/UAP

  • Observations récentes

  • Rencontres rapprochées

AMS

American Meteor Society

Organisation dédiée à l'étude des météores et bolides. Collecte systématique d'observations visuelles, calcul de trajectoires, identification d'origine.

Observations de météores

  • Bolides brillants (fireballs)
  • Trajectoires atmosphériques
  • Météoroïdes interstellaires
  • Pluies d'étoiles filantes

Comment utiliser ces liens avec ThibEquation ?

1. Compilez vos résultats avec les calculateurs ThibEquation (UFO Analytics, Premium, Standard).
2. Exportez vos données au format CSV, JSON ou Markdown via les boutons d'export.
3. Soumettez aux organismes appropriés selon le type d'observation :
   • Objets astrophysiques confirmés → JPL/NASA, MPC
   • Phénomènes UAP/UFO → MUFON, NUFORC
   • Météores/bolides → American Meteor Society
   • Datasets scientifiques → Zenodo (DOI automatique)
   • Publications académiques → arXiv, HAL
4. Citez ThibEquation Framework v5.0.0-EMPIRICAL (DOI: 10.5281/zenodo.17717380) dans vos soumissions.
5. Partagez vos résultats avec la communauté ThibEquation via [email protected]
